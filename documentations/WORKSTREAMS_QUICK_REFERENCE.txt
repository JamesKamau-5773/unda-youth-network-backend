╔══════════════════════════════════════════════════════════════════════════════╗
║                        HOW TO ACCESS (STEP-BY-STEP)                          ║
╚══════════════════════════════════════════════════════════════════════════════╝

Step 1: Login as Admin
   → Navigate to: http://127.0.0.1:5000/auth/login
   → Enter admin credentials
   → Click "Login"

Step 2: Access Workstreams Dashboard
   → In left sidebar, click "Workstreams" (grid icon)
   → You'll see 4 workstream cards

Step 3: Choose Your Action
   ┌─────────────────────────────────────────────┐
   │  FOR EACH WORKSTREAM CARD:                  │
   │  • Click "View All" → See all items         │
   │  • Click "Add New" → Create new item        │
   │  • On list page:                            │
   │    - Click "Edit" → Update item             │
   │    - Click "Delete" → Remove item           │
   └─────────────────────────────────────────────┘

╔══════════════════════════════════════════════════════════════════════════════╗
║                          EXAMPLE WORKFLOWS                                   ║
╚══════════════════════════════════════════════════════════════════════════════╝

 CREATE A NEW DEBATE EVENT:
   1. Workstreams → Debators Circle → "Add New"
   2. Fill form:
      - Title: "Youth Leadership Debate"
      - Date: 2026-02-20 14:00
      - Location: "Community Center"
      - Status: Upcoming
   3. Click "Create Event"
   4.  Event created!

 EDIT A PODCAST:
   1. Workstreams → Podcasts → "View All"
   2. Find podcast in list
   3. Click "Edit" button
   4. Update fields (e.g., change category)
   5. Click "Update Podcast"
   6.  Changes saved!

 DELETE A CAMPUS EVENT:
   1. Workstreams → Campus Edition → "View All"
   2. Find event in list
   3. Click "Delete" button
   4. Confirm deletion in browser
   5.  Event removed!

╔══════════════════════════════════════════════════════════════════════════════╗
║                           ACCESS CONTROL                                     ║
╚══════════════════════════════════════════════════════════════════════════════╝

 ADMINS CAN:
   • Access /admin/workstreams
   • Create, Read, Update, Delete ALL 4 workstreams
   • View statistics and filter results
   • Manage all content centrally

 SUPERVISORS/CHAMPIONS CANNOT:
   • Access workstreams dashboard
   • Modify any workstream content
   • See admin-specific routes
   • (Redirected if they try)

╔══════════════════════════════════════════════════════════════════════════════╗
║                         TECHNICAL IMPLEMENTATION                             ║
╚══════════════════════════════════════════════════════════════════════════════╝

Protection:  All routes use @login_required + @admin_required decorators
Database:    Events table (event_type distinguishes workstreams)
             Podcasts table (separate structure)
Templates:   Consistent UI patterns across all workstreams
Validation:  Event type checked on edit/delete to prevent cross-workstream edits
Flash msgs:  User feedback for all operations (success/error)
